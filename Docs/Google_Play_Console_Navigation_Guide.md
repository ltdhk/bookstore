# Google Play Console - API 访问权限导航指南

## 🔍 找不到"API 访问权限"？试试这些方法

### 方法 1：通过顶部搜索框（最快）

1. 登录 https://play.google.com/console/
2. 在页面顶部找到 **搜索框** 🔍
3. 输入：`API access` 或 `API 访问`
4. 直接点击搜索结果中的 **API 访问权限**

### 方法 2：通过所有应用页面

1. 登录 Play Console
2. 确保您在 **所有应用** 页面（不是在具体某个应用内）
3. 看左侧菜单，找到：

```
Play Console 主菜单：
├── 主页
├── 所有应用
│
├── 发布
│   ├── 制作
│   ├── 测试
│   └── 生产
│
├── 发布概览
│
└── 设置 ⚙️  ← 展开这个
    ├── 开发者账号
    ├── API 访问权限  ← 在这里！
    ├── 用户和权限
    └── 账号详细信息
```

4. 点击左侧菜单底部的 **设置** ⚙️
5. 展开后点击 **API 访问权限**

### 方法 3：直接访问 URL

如果您已经登录 Play Console，直接访问：

```
https://play.google.com/console/u/0/developers/[YOUR_DEVELOPER_ID]/api-access
```

**注意**：将 `[YOUR_DEVELOPER_ID]` 替换为您的开发者 ID，或者直接访问：

```
https://play.google.com/console/api-access
```

系统会自动跳转到正确的页面。

### 方法 4：通过设置菜单（2024年界面）

新版 Play Console 界面：

1. 点击右上角的 **账号图标** 或 **齿轮图标** ⚙️
2. 选择 **设置**
3. 在设置页面中，左侧菜单找到 **API 访问权限**

## 🚨 常见问题排查

### 问题 1：完全看不到"设置"菜单

**原因**：您可能在某个具体应用的页面内

**解决方案**：
1. 点击左上角的 **Play Console** 图标或 **所有应用**
2. 返回主菜单页面
3. 然后再找 **设置** 菜单

### 问题 2：看到"设置"但没有"API 访问权限"

**原因 A**：权限不足

您的账号必须是以下角色之一：
- ✅ 账号所有者
- ✅ 管理员
- ❌ 开发者、财务人员等其他角色 **看不到** API 访问权限

**解决方案**：
1. 联系 Play Console 账号所有者
2. 请求授予 **管理员** 权限
3. 或者请所有者帮您配置服务账号

**原因 B**：初次使用未激活

**解决方案**：
1. 访问：https://play.google.com/console/api-access
2. 首次访问可能需要接受服务条款
3. 点击 **接受** 或 **启用 API 访问**

### 问题 3：界面语言不是中文

如果界面是英文，找以下选项：
- `Settings` → `API access`
- 或顶部搜索 `API access`

## 💡 Pro 提示

### 快捷方式

1. **收藏 API 访问权限页面**：
   - 直接访问 https://play.google.com/console/api-access
   - 添加到浏览器书签

2. **使用键盘快捷键**（桌面版）：
   - `Ctrl + K` 或 `Cmd + K`：打开快速搜索
   - 输入 "API" 快速跳转

3. **检查控制台账号角色**：
   - 设置 > 用户和权限
   - 查看您的角色是否为 "管理员" 或 "所有者"

## 📞 仍然找不到？

### 选项 A：联系账号所有者

如果您不是账号所有者：
1. 联系 Play Console 账号所有者
2. 说明需要配置 IAP 验证
3. 请求 **管理员权限** 或请所有者帮您配置

### 选项 B：使用 Google Cloud Console

您也可以直接在 Google Cloud Console 中创建服务账号：

1. 访问：https://console.cloud.google.com/
2. 创建项目和服务账号
3. 下载 JSON 密钥
4. 在 Play Console 中稍后关联

然后按照 [Google_Service_Account_Setup_Guide.md](./Google_Service_Account_Setup_Guide.md) 中的步骤操作。

### 选项 C：提交支持工单

如果确实无法找到：
1. 访问：https://support.google.com/googleplay/android-developer/
2. 提交支持工单
3. 说明您需要访问 API 访问权限页面
